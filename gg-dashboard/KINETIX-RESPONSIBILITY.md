# Kinetix System Limited — Solution Architect Responsibility

**Role:** Solution Architect  
**Focus:** Security compliance, infrastructure operations, procurement, budget planning

---

## 🔐 Security & Compliance

- **Critical Infrastructure Law compliance** — design and implement security architecture that meets Hong Kong Critical Infrastructure (CI) Ordinance requirements
- **Security toolchain ownership** — lead deployment and configuration of:
  - **CyberArk** — privileged access management (PAM), credential rotation, session recording
  - **Tenable** — vulnerability scanning, risk scoring, remediation tracking
  - **CrowdStrike** — EDR/XDR endpoint protection, threat hunting, Falcon platform
  - **SOC** — security operations centre workflow, SIEM integration, incident response playbooks
- **Compliance audit** — prepare and present evidence for regulatory audits, maintain security documentation

## ⚙️ Daily Security Operations

- **Vulnerability management** — triage Tenable findings, assign severity, track SLA for remediation
- **Legacy system decommissioning** — assess end-of-life systems, plan migration paths, execute phased sunset
- **Platform maintenance** — keep Windows, Linux, VMware, and application estates patched and hardened
- **Incident response** — lead root cause analysis, implement containment, coordinate across infra and dev teams

## 📦 Procurement Lifecycle

- **End-to-end ownership** — from requirement gathering → vendor RFQ → technical evaluation → PO → delivery → acceptance
- **Vendor management** — negotiate licensing terms, evaluate alternatives, manage renewal calendar
- **Software asset management** — track entitlements, ensure compliance with licensing terms, avoid over/under-provisioning

## 💰 Budget & Planning

- **Infrastructure budget** — forecast hardware, software, cloud, and data centre costs for upcoming fiscal year
- **License renewal planning** — maintain forward calendar of all software renewals, optimise bundling and timing
- **Cost optimisation** — identify opportunities to consolidate, migrate to cloud, or eliminate redundant spend

## 🏗️ Architecture Deliverables

- **Solution design documents** — HLD, LLD for each project engagement
- **Technology roadmap** — align infrastructure evolution with business strategy and regulatory timeline
- **Proof of concept** — evaluate new technologies (e.g. SASE, Zero Trust, container security) against client requirements
