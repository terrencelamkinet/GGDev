**版本：** 3.0  
**適用年齡：** 1 至 40 歲  
**設備要求：** MacroTellect BrainLink 腦電波頭盔 + Windows 電腦  
**編制依據：** 美國兒科學會（AAP）、PMC 隨機對照試驗（n=104）、NHA 神經反饋指引

## 一、產品概覽

「專注飛鳥 Pro」是一款以神經反饋原理設計的認知訓練遊戲。玩家佩戴 BrainLink 腦電波頭盔，通過真實專注度數據控制螢幕上的飛鳥，專注度達到指定門檻時飛鳥下沉收集美食。遊戲共分 **10 層 × 10 關**，合計 100 關，食物主題逐層變換，難度循序漸進，支援 **1 至 40 歲**廣泛年齡段。

### 核心機制

- **腦電波下沉控制**：attention 值達到年齡對應門檻後，飛鳥以比例速度下沉。
- **EMA 平滑**：WebSocket 訊號以指數移動平均（α=0.18）處理，防止訊號抖動。
- **TBR 加成**：theta/beta 比值低於 2.5 時自動提升專注分數，鼓勵深度專注狀態。
- **年齡自適應難度**：1–40 歲共分 9 個年齡段，各段獨立設定速度、目標數量、重力與容錯值。
- **連擊獎勵**：連續收集觸發連擊加分，最高連擊提升音效音調與視覺特效，增強正強化。

## 四、操作說明

### 硬件連接步驟

1. 確認 BrainLink 頭盔已充電，並佩戴於額頭正中，調整至感測器貼緊皮膚。
2. 開啟 BrainLink 配套 App 或藍牙配對，確認訊號燈顯示綠色。
3. 在 Windows 電腦執行 Python 橋接程式：

```
python focus_bridge_windows.py --port COM5
```

（`COM5` 請替換為實際 COM 口，可用 `--scan` 指令查詢）

4. 訊號穩定後，在瀏覽器打開 `index.html`，左上角綠點亮起即代表已連接。
5. 在設定頁調整年齡滑桿，遊戲自動計算適合難度。
6. 按「開始遊戲」即可進入第 1 層第 1 關。

### 備用控制（無頭盔時）

- **空白鍵（Space）**：按住時飛鳥下沉，放開上升，用於測試遊戲流程。
- **觸控屏**：按住螢幕同效果。
- **滑鼠**：按住左鍵下沉。

## 五、一個月訓練計劃

本計劃參考以下學術研究：

- **美國兒科學會（AAP）**：神經反饋列為 ADHD 注意力訓練 Level 1 最強支持介入。
- **PMC 隨機對照試驗（2025）**（n=104）：每週 5 節 × 20 分鐘 × 4 週，顯著改善抑制控制，效果持續 6–12 個月。
- **Frontiers in Human Neuroscience（2014）**：每週 3–5 節 × 15 分鐘，2 週後觀察工作記憶改善。
- **NHA 指引（2025）**：10–20 節後大多數人感受改善，30–40 節效果穩固。

### 第一週：基礎適應

**目標**：建立遊戲習慣，成功率達 60% 以上

| 項目       | 說明                                  |
|------------|---------------------------------------|
| 訓練頻率   | 每週 3 天（例如：週一、三、五）        |
| 每節時長   | 8–12 分鐘（依年齡表）                 |
| 建議關卡   | 第 1 層 第 1–3 關                     |
| 下沉門檻   | 使用年齡預設值或調低 5–8 點           |
| 家長引導   | 陪伴，口頭鼓勵「好棒，小鳥飛下去了！」 |
| 成功指標   | 至少完成 2 關，孩子願意繼續玩          |

**神經科學依據**：初期建立「努力→成功→獎勵」的正強化迴路（AAP Level 1 原則）。成功率高於 60% 才能有效建立神經連結。

| 成功指標   | 每節出現 3 次以上連擊                  |

**神經科學依據**：Frontiers in Human Neuroscience（2014）研究顯示，兩週後工作記憶可出現可測量改善。連擊機制模擬「流態（Flow State）」的維持，對學習效能有正面影響。

## 六、每節訓練建議流程

1. **準備（2 分鐘）**：佩戴頭盔、深呼吸 3 次、告知孩子今天的目標關卡。
2. **暖身（3 分鐘）**：使用 Space 鍵玩 1 關，讓孩子熟悉操作感。
3. **正式訓練**：按計劃關卡進行，不中途強制繼續（孩子主動性優先）。
4. **休息（5 分鐘）**：訓練後休息，不立即看其他螢幕。
5. **記錄**：記下當日完成關卡數、最高專注度、孩子情緒反應。

---

## 七、注意事項

- **不強迫**：孩子表示疲倦或抗拒時立即停止，強迫訓練會降低效果。
- **安靜環境**：訓練期間減少周邊干擾（關電視、手機靜音）。
- **佩戴時間**：頭盔每次佩戴建議不超過 45 分鐘。
- **訊號品質**：額頭汗濕或頭盔移位會影響訊號，定期檢查綠點是否亮起。
- **年齡調整**：若孩子連續 3 節成功率低於 50%，可在設定頁降低門檻 5–10 點。
- **成人使用者**：26–40 歲建議每節訓練後休息 10 分鐘，避免腦部疲勞。

## 八、Windows Python 橋接程式說明

檔案：`focus_bridge_windows.py`

### 系統需求

- Python 3.8 或以上
- 安裝套件：`pip install websockets pyserial bleak`

### 常用指令

| 指令                                       | 說明                         |
|--------------------------------------------|------------------------------|
| `python focus_bridge_windows.py --scan`    | 掃描所有可用 COM 口           |
| `python focus_bridge_windows.py --port COM5` | 啟動橋接（COM5為範例）      |
| `python focus_bridge_windows.py --port COM5 --age 8` | 同時設定年齡（傳送至遊戲）|
| `python focus_bridge_windows.py --port COM5 --threshold 35` | 覆蓋門檻值      |

### 資料格式

程式將以下 JSON 資料廣播至 `ws://localhost:8765`：

```json
{
  "attention": 72,
  "meditation": 45,
  "signal": 0,
  "theta": 18000,
  "alpha": 12000,
  "highBeta": 8000,
  "timestamp": 1718121600000
}
```
