KONGGER Project Proposal

🎯 核心願景

KONGGER 是一個面向香港年輕世代的 Web 2.0 社交平台，靈感來自 Xanga 和 MySpace 的個人化精神，結合現代 Instagram/Threads 的社交互動，以及 Animal Crossing 的溫暖社群感。每個用戶都有一個完全屬於自己的數碼空間。

✨ 核心功能

  • 🏠
    Canvas 個人頁面
    Drag & Drop 裝飾你的個人空間，自訂佈局、主題和背景音樂
  • 📖
    心情日記
    附帶心情標記、媒體上傳、隱私設定的日記系統
  • 🎵
    BGM 播放器
    Spotify / YouTube / SoundCloud API 整合，個人化背景音樂
  • 🤖
    AI 手寫字體
    AI 將你的手寫上傳轉化為專屬 TTF 字體，在頁面中使用
  • 👥
    Neighbour System
    Dunbar 定律：每人最多 30 個鄰居，打造真實小圈子
  • 📢
    User-Owned Ad Space
    個人廣告版：你的流量，你的收益

🗺️ 開發路線圖

Phase 1
MVP Web
Figma UI → FlutterFlow → Firebase → Spotify 整合
Phase 2
Canvas Room
Drag & Drop 個人頁面 + 心情日記 + AI Font Generator
Phase 3
App + Monetization
Flutter App → Stripe/PayMe + App Store 上架
